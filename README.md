# Security Incident Mapping (SIM)

## Course
SEN202 – Software Engineering

## Project Title
Security Incident Mapping (SIM)

## Description
SIM is a simple web-based application that enables students, staff, and landlords to report security incidents occurring on or around campus.

## Features
- Home Page
- Report Security Incident
- View Reported Incidents
- Emergency Contacts

## Technologies Used
- HTML5
- CSS3
- JavaScript
